## Rules

- Capture quickly in `private/inbox/`
- Move reusable knowledge into the correct private folders
- Keep public content inside `public/`
- Do not publish directly from raw private notes
- Use templates for consistency
- Use markdown and frontmatter for every note
- Add research items to `checklist/read-later/` with tags and status checkbox
