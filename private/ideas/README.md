# Ideas

Startup ideas, product improvements, experiments, and side-project thoughts.

## Subfolders

- `startup-ideas/` - business and product concepts
- `product-improvements/` - ideas for existing products
- `experiments/` - tests and explorations

## Rules

- One idea per file
- Use the idea template
- Include problem, users, and next step
- Mark publish_candidate if worth sharing

## Format

```bash
# Filename example
saas-idea-client-portal.md
ai-content-pipeline-test.md
```
