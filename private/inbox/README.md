# Inbox

Fast capture area for rough notes.

## Rules

- Capture fast first, organize later
- One monthly file (e.g., `2026-04.md`)
- Review and clean at end of week or month
- Extract reusable knowledge into proper folders

## Structure

- `YYYY-MM.md` - monthly capture files

## When to use

- Quick notes
- Copied commands
- Links to revisit
- Half-formed ideas
- "fix this later" reminders
