# Setup: GitHub Copilot

GitHub's AI coding assistant, integrated directly into IDEs for inline completions and chat.

## Plugins / Packages

### VS Code Extensions

| Extension | Purpose | Install |
|-----------|---------|---------|
| GitHub Copilot | Inline code completions | VS Code marketplace |
| GitHub Copilot Chat | Chat panel + inline chat | VS Code marketplace |

### JetBrains

| Plugin | Purpose | Install |
|--------|---------|---------|
| GitHub Copilot | Completions + chat | JetBrains marketplace |

## Config

- Auth: `gh auth login` (uses GitHub CLI token)
- Settings: VS Code `settings.json` → `github.copilot.*`
