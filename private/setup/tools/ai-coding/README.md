## Comparison

| Tool | Type | Primary Use | IDE Support |
|------|------|-------------|-------------|
| Claude | CLI + desktop | Deep reasoning, long-context tasks, agentic work | VS Code ext, JetBrains ext |
| Copilot | IDE plugin | Inline completions, PR summaries, chat | VS Code, JetBrains, Neovim |
| Codex | CLI | Terminal-based coding tasks, OpenAI ecosystem | None (CLI only) |
| Opencode | CLI | Open-source alternative, self-hosted option | None (CLI only) |
